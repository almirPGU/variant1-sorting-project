## Задание

Программа работает со списком учеников 8–11 классов, и должна отсортировать список по заданным в задании ключам
Данные загружаются из файла `students.txt`.

Реализовано:
- чтение данных из файла;
- сортировка записей;
- вывод различных списков учеников;
- добавление ученика через консоль, в том числе и вручную;
- проверка корректности ввода;
- сохранение добавленных данных в файл;
- текстовое меню с возвратом назад.

Для сортировки используется сортировка вставками.
